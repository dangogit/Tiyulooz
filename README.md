# Tiyulooz

Hebrew travel guide app with AI assistant, trip planning, and curated guides.

## Tech Stack

- React Native, Expo SDK 52
- Expo Router (file-based navigation)
- TypeScript
- Zustand (state management)
- Supabase (backend and auth)
- NativeWind (Tailwind for React Native)
- Markdown rendering for guide content

## Features

- Curated travel guides with detailed content
- AI-powered travel assistant chat
- Trip planner with itinerary management
- User authentication and profile management
- Dark/light theme support
- Hebrew-first UI with custom Heebo fonts
- Onboarding flow with personalization

## Getting Started

```bash
git clone https://github.com/dangogit/Tiyulooz.git
cd Tiyulooz
npm install
npx expo start
```
